
# Simple startpage

Can be used "as is" or as a starting point for your customized homepage.

## Creating a project

Use the template button on github to start a new project

- create .env
- setup src/lib/colors.js
- change base in `svelte.config.js`
- run `yarn`
- run `yarn dev`
